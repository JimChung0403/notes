# 導入清單

## 1. 專案條件

- [ ] 專案可用單一命令啟動
- [ ] 專案有固定測試 URL
- [ ] 專案有健康檢查端點或可檢查首頁
- [ ] 專案可在測試環境產生 deterministic 資料
- [ ] 專案已有 smoke tests
- [ ] 專案已有 critical path tests

## 2. Playwright 條件

- [ ] 已安裝 Playwright
- [ ] 可輸出 HTML report
- [ ] 可輸出 trace / screenshot / video
- [ ] 已將 smoke 與 critical path 分組
- [ ] 測試可在 CI 執行

## 3. AI CLI 條件

- [ ] 已安裝 Qwen Code CLI 或 OpenCode CLI
- [ ] 可在目前環境無互動執行
- [ ] 已設定 API key 或 provider
- [ ] 已定義允許修改的路徑
- [ ] 已定義禁止修改的路徑

## 4. 安全與邊界

- [ ] 已設定最大自動修復輪數
- [ ] 已設定單輪最大 diff 規則
- [ ] 已設定環境問題的停止條件
- [ ] 已設定 secrets/infra 不可自動修改
- [ ] 已定義何時轉人工處理

## 5. 報告與追蹤

- [ ] 每輪都保存測試報告
- [ ] 每輪都保存應用日誌
- [ ] 每輪都保存 AI 修改摘要
- [ ] 每輪都能輸出 triage report

## 6. 建議預設值

- 最大自動修復輪數：`3`
- 快速守門：`lint + typecheck + unit`
- 第一層 E2E：`smoke`
- 第二層 E2E：`critical path`
- 停止條件：`同一類錯誤重複 2 次` 或 `超過 3 輪`

# 導入清單

## 1. 專案條件

- [ ] 可用單一命令啟動
- [ ] 有固定測試 URL
- [ ] 有健康檢查方式
- [ ] 可建立穩定的測試資料
- [ ] 至少已有 smoke tests
- [ ] 最好已有 critical path tests

## 2. 驗證條件

- [ ] 可跑 lint
- [ ] 可跑 typecheck
- [ ] 可跑 unit
- [ ] Playwright 可輸出 HTML report
- [ ] Playwright 可輸出 trace、screenshot、video

## 3. AI 條件

- [ ] 已安裝 Qwen Code CLI 或 OpenCode CLI
- [ ] 可在目前環境無互動執行
- [ ] provider 或 API key 已配置
- [ ] 已定義允許修改的目錄
- [ ] 已定義禁止修改的目錄

## 4. 安全邊界

- [ ] 已設定最大自動修復輪數
- [ ] 已設定單輪最大變更範圍
- [ ] 已定義環境問題的停止條件
- [ ] 已禁止修改 secrets、infra、production config
- [ ] 已定義何時轉人工處理

## 5. 建議預設值

- 最大輪數：`3`
- 快速守門：`lint + typecheck + unit`
- 第一層 E2E：`smoke`
- 第二層 E2E：`critical path`
- 停止規則：`同類錯誤重複 2 次` 或 `超過 3 輪`
